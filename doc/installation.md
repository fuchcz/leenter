# INSTALLATION
Easiest way to install Leenter is via Composer. File `composer.json` could look like this:

    {
        "repositories": [
            {
                "type": "vcs",
                "url": "git://github.com/fuchcz/leenter.git"
            }
        ],
        "require": {
            "fuchcz/leenter": "dev-master"
        }
    }
